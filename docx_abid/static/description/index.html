Export Odoo data to Microsoft Word (DOCX) and PDF

Design reports using Microsoft Office

Export data from any Odoo model to DOCX and PDF files using Word merge fields.

Report template (*.docx)

Output: DOCX or PDF

Word merge fields

Create a Word document and add merge fields. In Word: Insert → Quick Parts → Field → MergeField. Use field names like o.name, o.partner_id.name, etc.

In the Field dialog, choose MergeField and enter the field name (e.g. o.name). You will see <<o.name>> in the document.

Note

  • Do not insert images in the report template for merge.
  • Merge action does not fill field values in header & footer.
  • On Windows, PDF conversion may require unoconv/LibreOffice.
  • Tested on Odoo Community.
  • Conditional (if) logic in merge fields is not supported.

Python expressions

Global values & icons

Available in templates: env, user, company_id, lang, time, date, company_vat_label, company_name, company_footer_line, company_footer_line_icon, and icon placeholders (iphone, iemail, iwebsite, imoney, etc.).

Extra fields in res.partner & res.company

Includes: display_address, vat_label, data_line, data_line_icon, footer_line, footer_line_icon, and related variants. See Odoo field list for full details.
